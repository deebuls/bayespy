The VB lower bound and its gradients¶

The VB lower bound:

\[\begin{split}\mathcal{L} &= \underbrace{\langle \log p(X,Z) \rangle}_{\equiv \mathcal{L}_p} - \underbrace{\langle \log q(Z) \rangle}_{\equiv \mathcal{L}_q}\end{split}\]

The child nodes pass the gradient to the parent node so the parent node can optimize its parameters. In general, \(\mathcal{L}_p\) can be almost arbitrarily complex function of \(Z\):

\[\mathcal{L}_p = \langle \log p(X,Z) \rangle.\]

The gradient is

\[\begin{split}\frac{\partial}{\partial \xi} \mathcal{L}_p &= \frac{\partial}{\partial \xi} \langle \log p(X,Z) \rangle \\ &= \langle \log p(X,Z) \frac{\partial}{\partial \xi} \log q(Z) \rangle\end{split}\]

which can be computed, for instance, by sampling from \(q(Z)\). Note that \(\xi\) can represent, for instance, the expectation parameters \(\bar{u}\) of \(q(Z)\) in order to obtain the Riemannian gradient for an exponential family \(q(Z)\).

Often, \(\mathcal{L}_p\) has a simpler form (or it can be further lower bounded by a simpler form). If \(\mathcal{L}_p\) can be written as a function of \(\bar{u}\) as

\[\mathcal{L}_p = \bar{u}^T \psi + \mathrm{const},\]

the gradient with respect to the moments is

\[\frac{\partial}{\partial \bar{u}} \mathcal{L}_p = \psi.\]

It can be that \(\psi\) can be computed exactly by using moments of other nodes or it needs to be approximated by using sampling from the distribution of other nodes.

To summarize, the gradient message can be a numerical gradient, an approximate stochastic gradient (by sampling other nodes) or a function which can be used to compute an approximate stochastic gradient by sampling the node itself (and possibly other nodes).

Riemannian gradient¶

In principle, the VB lower bound can be maximized with respect to any parameterization of the approximate distribution. However, normal gradient can perform badly, because it doesn’t take into account the geometry of the space of probability distributions. This can be fixed by using Riemannian (i.e., natural) gradient. In general, the Riemannian gradient is defined as

\[\tilde{\nabla}_\xi \mathcal{L} = G^{-1} \nabla_\xi \mathcal{L}\]

where

\[[G]_{ij} = \left\langle \frac{\partial \log q(Z)}{\partial \xi_i} \frac{\partial \log q(Z)}{\partial \xi_j} \right\rangle = - \left\langle \frac{\partial^2 \log q(Z)}{\partial \xi_i \partial \xi_j} \right\rangle.\]

For exponential family distributions, the Riemannian gradient with respect to the natural parameters \(\phi\) can be computed easily by taking the gradient with respect to the moments \(\bar{u}\):

\[\tilde{\nabla}_\phi = G^{-1} \nabla_\phi \mathcal{L} = \nabla_{\bar{u}} \mathcal{L}.\]

Note that \(G\) depends only on the approximate distribution \(q(Z)\). Thus, the model itself does not need to be in the exponential family but only the approximation, in order to use this property. The Riemannian gradient of \(\mathcal{L}_q\) for exponential family distributions \(q(Z)\) is

\[\tilde{\nabla}_\phi \mathcal{L}_q = \nabla_{\bar{u}} \mathcal{L}_q = \nabla_{\bar{u}} [ \bar{u}^T \phi + \langle f(Z) \rangle + g(\phi) ] = \phi.\]

Thus, the Riemannian gradient is

\[\tilde{\nabla}_\phi \mathcal{L} = \nabla_{\bar{u}} \mathcal{L}_p - \phi.\]

Nonlinear conjugate gradient methods [1]:

  • Fletcher-Reeves:
\[\beta_n = \frac { \langle \tilde{g}_n, \tilde{g}_n \rangle_n } { \langle \tilde{g}_{n-1}, \tilde{g}_{n-1} \rangle_{n-1} } = \frac { \langle g_n, \tilde{g}_n \rangle } { \langle g_{n-1}, \tilde{g}_{n-1} \rangle }\]
  • Polak-Ribiere:
\[\beta_n = \frac { \langle \tilde{g}_n, \tilde{g}_n - \tilde{g}_{n-1} \rangle_n } { \langle \tilde{g}_{n-1}, \tilde{g}_{n-1} \rangle_{n-1} } = \frac { \langle g_n, \tilde{g}_n - \tilde{g}_{n-1} \rangle } { \langle g_{n-1}, \tilde{g}_{n-1} \rangle }\]
  • Hestenes-Stiefel:
\[\beta_n = - \frac { \langle \tilde{g}_n, \tilde{g}_n - \tilde{g}_{n-1} \rangle_n } { \langle \tilde{g}_{n-1}, \tilde{g}_{n-1} \rangle_{n-1} } = - \frac { \langle g_n, \tilde{g}_n - \tilde{g}_{n-1} \rangle } { \langle g_{n-1}, \tilde{g}_{n-1} \rangle }\]

where \(\langle \rangle_i\) denotes the inner product in the Riemannian geometry, \(\langle \rangle\) denotes the inner product in the Euclidean space, \(\tilde{g}\) denotes the Riemannian gradient and \(g\) denotes the gradient, and the following property has been used:

\[\langle \tilde{g}_n, \tilde{x} \rangle_n = \tilde{g}_n^T G_n \tilde{x} = g^T G^{-1}_n G_n \tilde{x} = g^T \tilde{x} = \langle g, \tilde{x} \rangle\]
