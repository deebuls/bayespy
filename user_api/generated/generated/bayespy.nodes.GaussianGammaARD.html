bayespy.nodes.GaussianGammaARD¶

class bayespy.nodes.GaussianGammaARD(mu, alpha, a, b, **kwargs)[source]¶

Node for Gaussian and gamma random variables with ARD form.

The prior:

\[p(x, \tau| \mu, \alpha, a, b) = p(x|\tau, \mu, \alpha) p(\tau|a, b)\]\[p(x|\alpha, \mu, \alpha) = \mathcal{N}(x | \mu, \mathrm{diag}( \boldsymbol{\alpha} \boldsymbol{\tau} ))\]\[p(\tau|a, b) = \mathcal{G}(\tau | a, b)\]

The posterior approximation \(q(x, \tau)\) has the same Gaussian-gamma form.

Warning

Not yet implemented.

See also

Gaussian, GaussianARD, Gamma, GaussianGammaISO, GaussianWishart
